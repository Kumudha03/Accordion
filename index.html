• HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages

  • CSS, or Cascading Style Sheets, is the language used to style and enhance HTML documents.

  • JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web.

  • Responsive web design is about creating web pages that look good on all devices! A responsive web design will automatically adjust for different screen sizes and viewports.
